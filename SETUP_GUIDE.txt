# HFTN House Updates Hub — Setup Guide
# Total time: ~20 minutes

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — SET UP SUPABASE (free database)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to https://supabase.com and click "Start your project"
2. Sign up with Google or email (free)
3. Click "New project"
   - Name it: hftn-hub
   - Set a database password (save it somewhere safe)
   - Choose region: Southeast Asia (Singapore) — closest to Manila
   - Click "Create new project" and wait ~1 minute

4. Once your project is ready, go to the SQL Editor (left sidebar)
5. Click "New query" and paste this ENTIRE block, then click Run:

─────────────────────────────────────
create table house_entries (
  id uuid default gen_random_uuid() primary key,
  house text not null,
  year integer not null,
  month integer not null,
  notes text,
  highlights text,
  prayer_points text,
  author text,
  photos jsonb default '[]',
  updated_at timestamptz default now(),
  unique(house, year, month)
);

alter table house_entries enable row level security;

create policy "Allow all access" on house_entries
  for all using (true) with check (true);
─────────────────────────────────────

6. Go to Project Settings → API (left sidebar)
   - Copy "Project URL" → this is your VITE_SUPABASE_URL
   - Copy "anon public" key → this is your VITE_SUPABASE_ANON_KEY
   - Save both — you'll need them in Step 3


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — GET YOUR ANTHROPIC API KEY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to https://console.anthropic.com
2. Sign in and go to "API Keys" in the left sidebar
3. Click "Create Key", name it "hftn-hub"
4. Copy the key (starts with sk-ant-...) — save it safely
   This is your VITE_ANTHROPIC_KEY

Note: You'll need credits on your Anthropic account.
The AI generation features use Claude API — very affordable for this usage.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — DEPLOY TO VERCEL (free hosting)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Option A — Easiest: Deploy from GitHub
---------------------------------------
1. Go to https://github.com and create a free account if you don't have one
2. Create a new repository called "hftn-hub" (set to Private)
3. Upload all the files from the hftn-hub folder you downloaded
4. Go to https://vercel.com and sign up with your GitHub account
5. Click "Add New Project" → Import your hftn-hub repo
6. Before clicking Deploy, click "Environment Variables" and add:
   - VITE_SUPABASE_URL = (paste your Supabase URL)
   - VITE_SUPABASE_ANON_KEY = (paste your Supabase anon key)
   - VITE_ANTHROPIC_KEY = (paste your Anthropic key)
7. Click Deploy — Vercel builds it automatically (~1 minute)
8. You get a URL like: https://hftn-hub.vercel.app 🎉

Option B — Drag and Drop (no GitHub needed)
--------------------------------------------
1. Open a terminal in the hftn-hub folder
2. Run: npm install
3. Run: npm run build
4. This creates a "dist" folder
5. Go to https://vercel.com → "Add New Project" → "Deploy without Git"
6. Drag the "dist" folder onto the Vercel upload area
7. Add your environment variables when prompted
8. Deploy!

STEP 4 — SHARE WITH YOUR TEAM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Just send your team the Vercel URL!
Example: https://hftn-hub.vercel.app

Everyone uses the same URL → all data goes to the same Supabase database.
The "Your Name" field in each entry shows who logged what.
